

This folder contains following codes and  sub-folders

					Codes

1) cmd.sh and path.sh are for setting the training commands and kaldi path...

2) Run_matlab_feats_extract_and_clean.sh - is used to extract the FDLP features at 400 Hz and dereverberate...

3) convert_2_kaldi.sh and dirList_kaldi are used to convert the FDLP features to kaldi readable format...



					Folders
					
1) conf - contains all the config files...

2) Input_raw_data - contains raw FDLP features at 400 Hz, to familiarize the folder 
   structure, see "Input_raw_data/train_clean_100_reverb_wpe/newfea"

3) data_prep - Contains codes to generate proper folder structure fot FDLP feature dereverberation...


4) local - contains some kaldi scripts...

5) LOGs - use this folder to save all the log files while submitting the jobs to the queue...

6) matlab_300_MB1_gn_arfit - contains MATLAB codes to extract FDLP features in 400 Hz resolution...

7) steps - contains some kaldi scripts...

8) clean_elp_estimation - contains codes for dereverberate the FDLP features using the trained model...

9) utils - contains some kaldi scripts...
