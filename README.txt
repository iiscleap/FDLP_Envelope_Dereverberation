

This folder contains the codes required to reproduce the work mentioned on CSL journal paper...

This folder has three sub folders...

1) ASR - Contains the codes required to run hybrid(HMM-DNN) ASR on VOiCES challenge ASR. Three possible set of features are there. a) The classic log-mel filter bank features b) FDLP features
	c) FDLP features after dereverbetation...

2) ENV_estimation - This folder contains codes to perform FDLP dereverberation.

3) Joint_modeling - This folder contains codes to perform FDLP dereverberation with joint training.
